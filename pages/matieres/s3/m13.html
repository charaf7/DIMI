Graphisme Dynamique

Format

Le cours se déroule en 06 séances de trois heures à partir du 30 janvier 2018.

Présentation

Dans ce cours nous étudions le web comme un médium visuel. Nous allons travailler sur les possibilités graphiques du web et réfléchir sur la manière de les rendre disponibles via des interfaces utilisateur. L’objectif est d’expérimenter les possibilités du Web pour la création et réalisation graphique. Les thèmes abordés sont :

  • Design graphique et médias numériques interactifs
  • Effets visuels comme paradigme de production
  • Design d’information
  • Programmation graphique
  • Technologies 3D
  • Tendances Web

Evaluation

  • La validation du cours se fait par contrôle continu
  • Les étudiants travaillent sur la conception et réalisation d’un devoir à rendre la dernière séance du cours : 13 mars 2017.
  • Les consignes du devoir seront présentés ponctuellement au cours des séances.
  • Le devoir peut être vu aussi comme une mini-réalisation ou comme une “expérimentation graphique et d’interface”.
  • Le devoir peut se faire de manière individuelle ou en équipe de deux personnes.
